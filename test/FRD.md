# FRD

1. Upload files to raw ingestion zone.
2. Stream events/logs to ingestion pipeline.
3. Validate, classify, and partition data.
4. Transform raw data to curated datasets.
5. Register metadata in catalog.
6. Query curated data with SQL.
7. Provide dashboard views for KPIs.
8. Trigger alerts on failures/SLA breaches.
9. Enforce role-based access and audit trail.
