# Requirement Definition (V2)

## Goal
Build an AWS-based data ingestion and analytics platform for multi-format data (CSV, Parquet, text, logs/events).

## Success Metrics
- 95% of ingested data queryable within 15 minutes
- Dashboard refresh <= 30 minutes
- Pipeline failure rate < 1% daily

## Scope
- Batch + streaming ingestion
- Raw to curated processing
- Cataloging + query + dashboards
- Security, observability, governance

## Out of Scope
- ML training pipelines
- Cross-cloud deployment

## Constraints
- AWS-first
- Cost-conscious managed services
